# 结构方程路径与《结构方程模型.docx》对比说明

## 一、当前代码中的路径设定

### 模型一（动机-情境双轮驱动）

**层级**：情境层(3) → 动机层(3) → 行为层(2)

| 类型 | 路径 | 对应假设 |
|------|------|----------|
| 情境→动机 | EEM ~ SMI + PSR + CTA | H1 SMI→EEM, H3 PSR→EEM, H6 CTA→EEM |
| | GBI ~ SMI + PSR | H2 SMI→GBI, H4 PSR→GBI |
| | RSA ~ PSR + CTA | H5 PSR→RSA, CTA→RSA（文档有城市文旅→仪式/自我实现） |
| 情境→行为(直接) | CTA → TWI | H7 城市文旅直接→旅游消费意愿 |
| 动机→行为 | PVI ~ EEM + GBI + RSA | H8–H10 |
| | TWI ~ EEM + GBI + RSA + CTA | H11–H13 + CTA 直接 |

**构念对应**：  
- 情境层：SMI（线上/社交媒体）、PSR（明星/准社会关系）、CTA（城市文旅）  
- 动机层：EEM（情绪价值）、GBI（群体认同）、RSA（仪式参与/自我实现）  
- 行为层：PVI（参与/观演意愿）、TWI（消费/旅游消费意愿）

---

### 模型二（动机-阻碍 SEM，含二阶 MOT）

**结构**：一阶动机(EEM, GBI, RSA) → 二阶 MOT；MOT + PCB → PVI, TWI

| 类型 | 路径 | 说明 |
|------|------|------|
| 二阶因子 | MOT =~ EEM + GBI + RSA | 内在动机由三个一阶动机反映 |
| 结构 | PVI ~ MOT + PCB | 动机(正) + 阻碍(负) → 观演意愿 |
| | TWI ~ MOT + PCB | 动机(正) + 阻碍(负) → 旅游消费意愿 |

**构念**：EEM, GBI, RSA, PCB（感知成本障碍）, PVI, TWI，+ 二阶 MOT。

---

## 二、与《结构方程模型.docx》的对应关系

### 方案一（文档） vs 模型一（代码）

- 文档：**情境层(3) → 动机层(3) → 行为层(2)**，“情景层-动机层-行为层两阶段探究意愿”。  
- 代码：层级与构念划分一致；路径为  
  - 情境→动机：EEM/GBI/RSA 分别由 SMI、PSR、CTA 回归；  
  - CTA→TWI 直接效应；  
  - 动机→意愿：PVI、TWI 由 EEM、GBI、RSA（及 TWI 再加 CTA）回归。  

**结论**：**一致**。文档中的“两阶段”“几条路径”与代码中的模型一设定相符；若文档中图 1 是“情境→动机→行为”且含 CTA 直连 TWI，与代码一致。

---

### 方案二（文档） vs 模型二（代码）

- 文档方案二：  
  - **动机**：情绪价值(EM)、群体认同(SI)、仪式参与(RP) → 对应代码 EEM、GBI、RSA。  
  - **阻碍**：文档列出**两个**阻碍因子——**4. 成本价值阻碍(CB)**、**5. 服务体验顾虑(SE)**（SE1/SE2/SE3：入场散场混乱、交通疏散、城市不友好）。  
  - **行为意愿**：参与意愿(PI)、购买/消费意愿(PU) → 对应代码 PVI、TWI。  

- 代码与问卷现状：  
  - 动机：MOT =~ EEM + GBI + RSA，与文档一致。  
  - **阻碍**：仅有 **PCB（感知成本障碍）**，对应文档中的 **CB（成本价值阻碍）**；问卷中 Scale_7 三题均为“金钱成本/精力负担/住宿餐饮溢价”，**没有“服务体验顾虑(SE)”的题项**。  
  - 行为：PVI、TWI 与文档 PI、PU 对应一致。  

**结论**：  
- **动机 + 行为意愿部分**：与文档**一致**。  
- **阻碍部分**：文档为 **CB + SE 两个因子**，代码与问卷仅实现 **PCB（=CB）一个因子**，**缺少 SE 及 SE→PVI、SE→TWI 的路径**。若《结构方程模型.docx》中的方案二图里画了“CB 与 SE 两条阻碍路径”，则与当前代码**不完全一致**；若图中只画了一个“阻碍”或只画了成本阻碍，则与代码一致。

## 三、差异汇总与建议

| 项目 | 文档 | 代码/问卷 | 是否一致 |
|------|------|-----------|----------|
| 方案一 层级 | 情境→动机→行为 | 同左 | ✓ 一致 |
| 方案一 情境→动机 | SMI/PSR/CTA → EEM/GBI/RSA | 同左（EEM~SMI+PSR+CTA 等） | ✓ 一致 |
| 方案一 CTA→TWI 直接 | 两阶段中“消费”可受情境影响 | CTA→TWI 有路径 | ✓ 一致 |
| 方案一 动机→意愿 | EEM/GBI/RSA → PVI/TWI | 同左 | ✓ 一致 |
| 方案二 动机 | EM/SI/RP → MOT | EEM/GBI/RSA → MOT | ✓ 一致 |
| 方案二 阻碍 | **CB + SE 两个因子** | **仅 PCB(CB)，无 SE** | ⚠ 不一致 |
| 方案二 意愿 | PI, PU | PVI, TWI | ✓ 一致 |

**建议**：  
- 若要以文档图为准且保留“双阻碍”：需在问卷中增加**服务体验顾虑(SE)** 的题项（如与入场秩序、散场疏散、城市友好度相关），并在模型二中增加潜变量 SE 及路径 SE→PVI、SE→TWI。  
- 若维持当前问卷（仅成本障碍）：可在正文中说明“本研究方案二仅纳入**感知成本障碍(PCB)** 作为阻碍因子，服务体验顾虑(SE) 留待后续研究”，并在图中只画 PCB→PVI、PCB→TWI，与代码一致。
